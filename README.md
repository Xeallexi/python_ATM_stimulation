# 简易 ATM 银行系统（Python 纯手工版）

一个使用 **原生 Python** 完全从零编写的银行 ATM 模拟器  
2025.11.30 凌晨独立完成，无任何框架依赖

## 项目特色
- 完整的用户交互菜单
- 支持查询余额、存款、取款、退出
- 完善的输入校验与错误处理（余额不足、非法输入）
- 使用 `global` 实现余额持久化
- 代码结构清晰、注释完整、可直接用于教学或简历展示
